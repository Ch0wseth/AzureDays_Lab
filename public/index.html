🧠 Fonctionnalités Copilot présentées

  • Service CRUD complet - taskService.js généré avec JSDoc
  • Routes Express - Endpoints REST avec validation et gestion d'erreurs
  • Utilitaires - Fonctions de validation, sanitisation, pagination

  • Ask Mode - Poser des questions sur le code, obtenir des explications
  • Edit Mode - Refactoring, ajout de fonctionnalités inline
  • Agent Mode - Tâches complexes multi-fichiers avec plan d'exécution

  • Custom Instructions - .github/copilot-instructions.md
  • Prompt Files - .github/prompts/*.prompt.md
  • MCP Playwright - Tests E2E via serveur MCP
  • Custom Agent - Agent spécialisé Orange Design System

  • .copilotignore - Exclure fichiers non pertinents du contexte
  • Instructions ciblées - Prompts précis pour réduire les allers-retours
  • Fichiers .prompt.md - Réutiliser les prompts fréquents
  • Vision Files - #file pour limiter le contexte envoyé
